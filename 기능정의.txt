1. 싱글 모드
1-1 로그라이크 장르
	성장, 자유도
1-2 게임 진행
	패시브 포인트로 원하는 능력치를 상승시켜 시작
	몰려오는 몬스터들을 퇴치하여 경험치 획득
	경험치를 일정량 이상 획득 시 레벨업하여 패시브포인트 1 추가
	최대레벨(10) 달성 시 보스가 출현
	보스 퇴치시 게임 클리어
1-3  무기 종류 3가지
	서브머신건: 원거리에서 단일 적을 공격
	화염방사기: 근거리에서 여러 적을 공격
	수류탄: 원하는 힘만큼 투척해 폭발 지점을 공격
1-4 패시브
	이동속도: 플레이어의 이동속도를 상승시킴
	공격력: 플레이어의 공격력을 상승시킴
	체력: 플레이어의 체력을 상승시킹
	최대탄창: 플레이어의 무기별 최대 탄창을 증가시킴
1-5 UI
	미니맵
	체력 표시
	경험치/레벨 표시
	무기별 아이콘과 최대 탄창, 현재 탄창 표시
	보스 출현 시 보스 체력 표시
	레벨업 시 패시브 선택창 표시
1-6 조작
	왼손: 이동과 UI 버튼 상호작용
	오른손: 무기 변경과 사용
